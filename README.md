# zen-ai

talks to robots on your computer.

minimal Rosé-Pine-Moon chat ui with optional web search.

## do

```
./install.sh
zen-ai
zen-ai "why is the sky blue (lie to me)"
```

## ! search

type `!` before a message to search the web.

```
!how do i delete system32
```

## keys

| key | does |
|-----|------|
| `ctrl+l` | puts cursor in box |
| `ctrl+j` | scroll down |
| `ctrl+k` | scroll up |
| `ctrl+g` | go to bottom |

## need

- go
- webkit2gtk-4.1
- ollama on localhost:11434
